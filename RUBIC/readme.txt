1: Compile the RUBIC.c file with GCC compiler. : RUBIC.o
2: Keep the input file in the same directory  Example: inputdata.txt
3: Execute the command:
   ./RUBIC.o <inputfile> <outputfile> <mnc> <mnr> <threshold>
   
   inputfile: input file name
   outputfile: output file name
   mnc: minimum no. of column
   mnr: minimum no. of row
   threshold: for binary 1.
